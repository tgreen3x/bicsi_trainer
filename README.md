This is a simple test prep and question generator in order to help a user prepare for the BICSI Technician licensing exam.
I created this in an effort to produce high quality, helpful question free of charge.
There is also a small flashcards section, a daily tracker, and a rewording lab.

The pressure round question generator is adaptive, and will learn from your mistakes which types of questions to pull from the bank.

The bank itself is a JSON array bank that a query can pull from and randomize. All questions are stored by ID and contain a section of the book
that relates to the question, allowing the user to get back into their books for any problems or misunderstandnings that arise.

Everything is written in HTML.
